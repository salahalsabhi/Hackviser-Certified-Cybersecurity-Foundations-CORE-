# Overview

This repository documents my successful completion of the **Hackviser Certified Cybersecurity Foundations (CORE)** certification.

The CORE certification is an entry-level cybersecurity program designed to help learners build practical security knowledge through structured learning modules, guided labs, and hands-on exercises.

Unlike certifications that focus solely on theory, this course combines cybersecurity concepts with practical scenarios, introducing learners to networking, cryptography, reconnaissance, threat management, web technologies, OSINT, and modern AI security.

The course also introduces emerging topics such as **LLM (Large Language Model) Security**, making it one of the first beginner certifications to include AI security fundamentals.

# Key Takeaways

Some of the most valuable lessons from this certification include:

- Security begins with strong fundamentals.
- Understanding networking is essential before learning offensive security.
- Cryptography underpins almost every secure communication system.
- OSINT can reveal significant information without directly interacting with a target.
- AI introduces entirely new attack surfaces that require new defensive strategies.
- Prompt Injection is becoming one of the most important AI security threats.
- Hallucinations are an inherent characteristic of LLMs and require verification.
- Ethical and legal boundaries are fundamental in cybersecurity testing.
